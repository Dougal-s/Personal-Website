About

Vkav is a highly configurable audio visualiser written in c++ for Linux.
Using Vulkan, it visualises audio from the desktop, or a user specified source, in real time.

*Windows and MacOS support had not been implemented*

Free and Open Source

Vkav is free and open source, meaning that you are free to download, redistribute and modify the source code. Futhermore, the software license allows you to use Vkav in commercial and/or proprietary software.

The license can be found here.

Highly Customizable

Config files allow you to tweak anything from sample rates and buffer sizes to sensitivity and colour. The selection of built in modules allow you to pick from several unique visuals. If you can't find one to your taste then you can always make your own module by following the tutorial here.

Contribute

Development is currently active and reporting bugs is encouraged. Bugs and features can be reported/requested here.

If you are a developer, you can contribute to or fork the github repository.
